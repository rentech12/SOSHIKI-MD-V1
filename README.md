# SOSHIKI-WEB ⚡

Générateur de SESSION-ID pour **SOSHIKI-MD-V1** par 𝐌ꝛ 𝐑𝐄𝐍 𝚨𝚻𝚳𝚵

## 🚀 Déploiement sur Koyeb

1. Push ce repo sur GitHub : `github.com/rentech12/soshiki-web`
2. Va sur [Koyeb](https://app.koyeb.com)
3. **New App** → **GitHub** → sélectionne `soshiki-web`
4. Builder : **Dockerfile**
5. Port : **3000**
6. Nom de l'app : `soshiki-web` → l'URL sera `soshiki-web.koyeb.app`
7. Deploy !

## 🖥️ Lancer en local

```bash
npm install
node index.js
# Ouvre http://localhost:3000
```

## ✨ Fonctionnalités

- 📱 **Code Pairing** — Entre ton numéro et reçois un code à 8 chiffres
- 📷 **QR Code** — Scanne avec WhatsApp
- SESSION-ID envoyée directement sur ton WhatsApp
- Interface dark mode premium
